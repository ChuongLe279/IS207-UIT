Commit vui lòng bằng tiếng anh để dễ xử lí

- Cú pháp: 

`type_commit(tên file hoặc nếu nhiều tên file thì ghi tên module chính): thay đổi`

- Commit nếu thêm tính năng mới

Ví dụ: `feat(board): export image`

- Commit nếu sửa gì đó

Ví dụ: `fix(board): export image`

- Commit nếu thay đổi không phải tính năng hay là sửa gì, thêm comment vào trong code, hoặc xoá gì đó không ảnh hưởng gì cả, ...

Ví dụ: `chore(api): remove abcxyz`

- Commit dành riêng cho front-end dev về việc thay đổi 1 chỗ nào đó trong giao diện

Ví dụ: `style(button): change color from red to blue`

- Commit cho docs

Ví dụ: `docs(api): update description`

- Commit nếu optimize code hay gì đó

Ví dụ: `enhance(api): optimize to O(n)`

- Commit nếu như thay đổi cấu trúc source code, đơn giản như format lại code 1 tí, ...

Ví dụ: `refactor(api): seperate api into 2 files`
